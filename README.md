# Portdict

Portdict uses python 3, `beautifulsoup` and `requests` to find the definition of ports.

Usage:  `portdict [port number]`

##### Example output:
```
$ portdict [port]

Port:  [port]
Protocol: [protocol]
Service: [service]
Description: [discription]
```

##### Requirements
For this to work you need:
- Beautiful Soup 4
- `requests` pip module
